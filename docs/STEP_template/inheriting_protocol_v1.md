Please follow the structure and guidelines to fill in the fields and write your protocol. 
You can use markdown formatting in the description and steps.

## Supplementary Material

- If you want to make a new supplementary section, you can just add it here, and it will override the one in the original protocol. 
- If you want to have the same supplementary as the original protocol, just delete the supplementary section here, and it will keep the original one.
- Same applies to the references. For example here I comment it out so that we keep the original references, but if I want to modify the references I can just write them here and it will override the original ones.
