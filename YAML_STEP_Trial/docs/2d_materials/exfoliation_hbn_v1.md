# hBN exfoliation protocol

**Authors:** Jaime Díez Mérida
**Created:** 2026-01-27
**Last modified:** 2026-01-30

## Description

This STEP protocol defines the process parameters for exfoliating graphene flakes. The protocol is described following Huang et al. ACS Nano (2015).


# Substrate preparation

| Step | Task | Subtask | Materials | Parameters | Warnings | Verification |
|------|------|-------|------------|----------|--------------|---------|
| 1.0 | Substrate preparation |  | Si/SiO2 chips, tweezers, O2 plasma cleaning machine, IPA, acetone, glass beakers. Wafers from Bonda etc. | - SiO2 ~285 nm (±10 nm). Preferred 90 nm for thin hBN. Prefer thermal oxide. | Acetone and IPA hazardous → use fumehood. | Clean chips, no particles or stains. |
| 1.1 |  | Acetone cleaning | Acetone, beaker, tweezers, optional sonicator | 10 s – 5 min depending on contamination | Clean beaker beforehand. Final rinse with same solvent. | Clean surface |
| 1.2 |  | IPA cleaning | IPA, beaker | Similar to acetone step | Same considerations as above | Clean chips |
| 1.3 |  | Drying | N2 gun | Dry thoroughly | Poor drying → stains. Tweezers can leave droplets. | Clean chips under microscope |
| 1.4 |  | Plasma treatment | Plasma asher / RIE | Example recipe: 100 W, 3 min, P = 0.2–0.5. Use within 10–20 min. | Too strong → rough surface. Optimize locally. | Better exfoliation vs untreated chips |
