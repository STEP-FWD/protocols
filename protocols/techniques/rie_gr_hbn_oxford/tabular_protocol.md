# Description

This STEP protocol describes reactive ion etching (RIE) of two-dimensional materials using SF6-, O2-, and CHF3-based recipes on an Oxford Plasmalab 100. The processes are tailored for small samples on silicon carrier chips (~5 mm). *(This file is a short test excerpt from the source presentation; the full protocol continues in `STEP_RIE_Oxford_hBN_Gr.pptx` in this folder.)*
